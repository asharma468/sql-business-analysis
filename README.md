# SQL Business Analysis Project

This project explores a fictional retail dataset to answer common business questions using SQL.

## Business Questions
- How many customers and orders does the business have?
- Which products generate the most revenue?
- Who are the highest-value customers?
- What trends can be observed in purchasing behavior?

## Tools Used
- SQL
- SQLite (or MySQL/PostgreSQL)

## Outcome
The analysis demonstrates how SQL can be used to support business and product decisions through data.
